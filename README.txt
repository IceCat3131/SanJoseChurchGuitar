将你的 SVG 放到以下目录中：

- 大本诗歌（C 开头）：assets/c/
  例如：assets/c/c0001_cn_g.svg, assets/c/c0788_cn_g.svg

- 小本诗歌（TS 开头）：assets/ts/
  例如：assets/ts/ts0001_cn_g.svg, assets/ts/ts0788_cn_g.svg

文件命名规则：
- 大本：c + 四位数字 + "_cn_g.svg"
  如：c0001_cn_g.svg
- 小本：ts + 四位数字 + "_cn_g.svg"
  如：ts0001_cn_g.svg

网站使用方式：
1. 将整个文件夹上传到任意静态空间（比如 GitHub Pages / Cloudflare Pages）
2. 访问 index.html 即可看到曲目列表
3. 点击某一首即可进入查看页面，自动加载对应 SVG
